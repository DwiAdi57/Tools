# 📊 Analisa Saham MERCY-BAJAI

Aplikasi analisa saham komprehensif yang menggabungkan Strategi Fundamental (Lo Kheng Hong / Buffett) dan Analisa Teknikal (TradingView style).

> *"Cari Saham Mercy di Harga Bajai"* — Temukan perusahaan berkualitas tinggi (Mercedes) yang dijual dengan harga murah (Bajaj).

## 🚀 Fitur Utama
- **Hybrid Scoring**: Menggabungkan fundamental (60%) dan teknikal (40%) menjadi satu "Master Power Score".
- **Multi-Market**: Mendukung IDX (Indonesia), Wall Street (US), dan Cryptocurrency.
- **Visualisasi Premium**: Grafik Candlestick interaktif dengan MA50, MA200, dan RSI.
- **Mode Pro**: Indikator teknikal lanjutan seperti Bollinger Bands dan valuasi DCF.
- **Simulasi Portfolio**: Fitur paper trading untuk mencatat simulasi pembelian Anda.
- **Master Table**: Screening dan perbandingan multi-saham sekaligus.

## 🛠️ Cara Menjalankan (Lokal)
1. Pastikan Python terinstall (v3.9+).
2. Install dependensi:
   ```bash
   pip install -r requirements.txt
   ```
3. Jalankan aplikasi:
   ```bash
   streamlit run app.py
   ```

## ☁️ Deploy ke Streamlit Cloud
1. Push repository ini ke GitHub.
2. Buka [share.streamlit.io](https://share.streamlit.io).
3. Hubungkan repository, pilih `app.py` sebagai main file.
4. Klik **Deploy**!

## 📦 Dependensi
- `streamlit` — Framework web app
- `yfinance` — Data saham real-time
- `pandas` — Manipulasi data
- `plotly` — Visualisasi interaktif
- `numpy` — Komputasi numerik
